# ITSK25 skriptimise lõputöö

Selles repos on kaks skripti: üks Pythonis ja teine PowerShellis.

## Andmete generaator (Python)
Fail: ITSK25_regnar_generator.py

Selles skriptis:
- ma küsin kasutajalt maatriksi suuruse (3 kuni 10)
- ma genereerin juhuslikud numbrid vahemikus 1 kuni 99
- uhekoha numbrid algavad nulliga
- ma salvestan tulemuse faili output.txt

## Andmete analuusaator (PowerShell)
Fail: ITSK25_regnar_analyzer.ps1

Selles skriptis:
- ma loen faili output.txt
- ma naitan maatriksi konsoolis
- ma arvutan iga rea ja veeru summa
- ma leian suurima ja vaikseima numbri

## Kaivitamine
1. kaivitan esmalt Python skripti
2. see loob faili output.txt
3. seejaarel kaivitan PowerShell skripti
